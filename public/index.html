Hướng dẫn cài đặt
1
Cấu hình file .env
Điền đầy đủ thông tin vào file .env: Plane API Token, Workspace Slug, và Discord Webhook URL.
2
Expose server ra internet (dùng ngrok hoặc tương tự)
Plane cần URL công khai để gửi webhook. Dùng: ngrok http 3000 → lấy URL dạng https://xxxx.ngrok.io
3
Cấu hình Webhook trên Plane.so
Vào Settings → Webhooks trong workspace Plane → Thêm URL: https://your-url.ngrok.io/webhook → Chọn event Issues
4
Test kết nối
Dùng các nút test ở trên để xác nhận Discord và Plane API đều hoạt động, sau đó tạo thử một work item trên Plane để xem thông báo!
